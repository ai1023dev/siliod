IP, DNS, 포트 개념

네트워크의 세계에서 IP, DNS, 포트는 인터넷 통신의 핵심 요소들입니다. 이들을 이해하면 인터넷이 어떻게 작동하는지 알 수 있습니다.

IP 주소는 인터넷상의 각 컴퓨터가 가지는 고유한 주소입니다. 마치 우편물을 보낼 때 사용하는 집 주소와 같습니다. 192.168.1.1처럼 점으로 구분된 네 개의 숫자로 이루어져 있습니다. 인터넷에 연결된 모든 기기는 이런 IP 주소를 가지고 있어서, 이 주소를 통해 서로 통신할 수 있습니다. 집에서 사용하는 공유기도 IP 주소를 가지고 있고, 여러분의 스마트폰이나 노트북도 각각 고유한 IP 주소를 받아서 인터넷을 사용합니다.

DNS는 도메인 이름 시스템으로, IP 주소를 사람이 기억하기 쉬운 이름으로 바꿔주는 서비스입니다. google.com이나 naver.com처럼 우리가 평소에 사용하는 웹사이트 주소가 바로 도메인 이름입니다. 실제로는 구글도 특정 IP 주소를 가지고 있지만, 숫자로 된 IP 주소를 외우기는 어렵기 때문에 DNS가 이를 도메인 이름으로 변환해줍니다. 마치 전화번호부에서 이름으로 전화번호를 찾는 것과 같은 원리입니다.

포트는 하나의 컴퓨터 안에서 여러 서비스를 구분하는 번호입니다. 한 건물에 여러 호실이 있는 것처럼, 하나의 IP 주소를 가진 컴퓨터에서도 여러 서비스가 동시에 실행될 수 있습니다. 웹서버는 보통 80번 포트를, 이메일 서버는 25번 포트를 사용합니다. 웹사이트에 접속할 때 www.example.com:80처럼 포트 번호를 명시할 수 있지만, 웹은 기본적으로 80번 포트를 사용하기 때문에 생략해도 됩니다.
