리눅스 터미널에서 자주 사용하는 기본 명령어들을 익히면 컴퓨터를 훨씬 효율적으로 사용할 수 있습니다. 이 명령어들은 파일과 폴더를 관리하는 기본적인 도구들입니다.

ls 명령어는 현재 위치의 파일과 폴더 목록을 보여줍니다. 윈도우 탐색기를 열어서 파일들을 보는 것과 같은 역할입니다. ls -l을 입력하면 파일의 자세한 정보까지 함께 볼 수 있습니다.

cd 명령어는 폴더를 이동할 때 사용합니다. cd Documents라고 입력하면 Documents 폴더로 이동합니다. cd ..는 상위 폴더로 이동하고, cd만 입력하면 사용자의 홈 폴더로 돌아갑니다.

cp는 파일을 복사하는 명령어입니다. cp file1.txt file2.txt라고 입력하면 file1.txt를 복사해서 file2.txt라는 이름으로 만듭니다. mv는 파일을 이동하거나 이름을 바꿀 때 사용합니다. mv old.txt new.txt는 파일 이름을 바꾸고, mv file.txt /home/user/는 파일을 다른 폴더로 이동시킵니다.

rm은 파일을 삭제하는 명령어입니다. rm file.txt라고 입력하면 해당 파일이 삭제됩니다. 주의할 점은 리눅스에서는 휴지통이 없어서 삭제된 파일은 복구하기 어렵습니다. mkdir은 새 폴더를 만드는 명령어입니다. mkdir newfolder라고 입력하면 newfolder라는 이름의 폴더가 생성됩니다.

sudo는 관리자 권한으로 명령어를 실행할 때 사용합니다. 시스템 파일을 수정하거나 프로그램을 설치할 때 필요합니다. sudo apt install firefox처럼 다른 명령어 앞에 붙여서 사용합니다.
